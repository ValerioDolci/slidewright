Come si usa

Gesti

  • Doppio click su un testo → modifica
  • Trascina un elemento → sposta snap automatico · Alt = libero
  • Maniglie → ridimensiona Shift = mantieni proporzioni
  • Trascina un file .html nella finestra → importa deck

Scorciatoie

  • ⌘S salva sul file
  • ⌘Z / ⇧⌘Z annulla / ripeti
  • ⌘C / ⌘V / ⌘D copia / incolla / duplica
  • ← ↑ → ↓ sposta 1px Shift = 10px
  • Canc elimina · Esc deseleziona
